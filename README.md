# সিলেবাসের বাইরে | Make Education Simple

This is a React-based educational platform that helps students prepare for their exams with easy explanations and modern techniques.

## Features

- Course listings with detailed information
- Authentication system
- Responsive design for all devices
- Interactive UI elements
- Course enrollment system

## Installation

1. Clone the repository
2. Navigate to the project directory
3. Install dependencies: `npm install`
4. Start the development server: `npm start`

## Available Scripts

In the project directory, you can run:

### `npm start`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

The page will reload when you make changes.\
You may also see any lint errors in the console.

### `npm test`

Launches the test runner in the interactive watch mode.\
See the section about [running tests](https://facebook.github.io/create-react-app/docs/running-tests) for more information.

### `npm run build`

Builds the app for production to the `build` folder.\
It correctly bundles React in production mode and optimizes the build for the best performance.

The build is minified and the filenames include the hashes.\
Your app is ready to be deployed!

## Project Structure

```
src/
├── components/         # Reusable React components
│   ├── Header.js       # Navigation header
│   ├── Hero.js         # Hero section
│   ├── ActionButtons.js # Action buttons section
│   ├── CoursesSection.js # Courses listing
│   ├── Footer.js       # Page footer
│   └── EnrollModal.js  # Enrollment modal
├── App.js             # Main application component
├── App.css            # Global styles
├── index.js           # Application entry point
└── index.css          # Base styles
```

## Dependencies

- React: JavaScript library for building user interfaces
- React DOM: Provides DOM-specific methods
- React Scripts: Configuration and scripts for Create React App
- React Router DOM: Routing library for navigation
