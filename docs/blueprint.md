# **App Name**: Hello World App

## Core Features:

- Display 'Hello World': Displays the classic 'Hello, World!' message.

## Style Guidelines:

- Primary color: Calm blue (#64B5F6) for a simple and clean feel.
- Background color: Light gray (#EEEEEE) to provide a neutral backdrop.
- Accent color: Soft green (#A5D6A7), an analogous hue, for focus/call to action.
- Font: 'Inter', a grotesque-style sans-serif font suitable for headlines or body text.
- Simple centered layout.
